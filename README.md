# BlockPicker
Tap the block to pick it to your inventory

## Command and Permission
- Command: `/blockpicker`
  - Alias: `/pickblock`
  - Permission: `blockpicker.command`
    - Default: `op`

## Usage
Start and join to your server, then do:
1. Set the `blockpicker.command` permission to use the command
2. Execute command `/blockpicker`
3. Tap a block

## Notice
You need to switch to creative mode to pick the block.

Set `betterBlockPick` in the config to `true` if `BetterBlockPick` plugin is installed in your server.

## Optional plugins
For Vanilla-like block picking system:
- [VanillaPickBlock by PresentKim](https://poggit.pmmp.io/p/VanillaPickBlock)
- [BetterBlockPick by Matze997](https://poggit.pmmp.io/p/BetterBlockPick)
